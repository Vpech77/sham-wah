# 🐐 Sham-Wah 🐐

Sham‑Wah is a web application currently being developed for the [IntForOut Project](https://www.umr-lastig.fr/intforout/). Its aims to make it easier to share and explore many resources (scientific papers, datasets, scripts, services, and more) produced and used by the project’s research teams.

## 🚧 Work in Progress 🚧

The application provides an initial layout prototype, displaying a query panel on the left with a view of the selected node’s neighbors on the right. At this stage, it is a visual demonstration only: the data shown is simulated and always appears when the Search button is used.

My main focus for now, has been on the predefined query panel, which highlights digital assets associated with human activities
